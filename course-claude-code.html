Программа курса

5 блоков — от установки до полного автопилота

01
Старт с Claude Code — установка и первые команды
Настраиваем окружение и делаем первое задание через диалог
›

Устанавливаем Claude Code — это npm-пакет, работает через терминал. Разбираем что такое терминал для тех кто никогда не открывал: это просто текстовый интерфейс где ты пишешь команды. Claude Code делает большинство команд за тебя — ты просто описываешь задачу. Настраиваем API-ключ от Anthropic (у тебя есть Pro-подписка — покажем как получить ключ). Делаем первое задание: просим Claude Code создать простую HTML-страницу и открыть её в браузере.

Установка Node.js npm install API ключ Anthropic Первые команды
02
Автоматизация сайта — генерация страниц из данных
JSON с курсами → 10 готовых HTML-страниц за одну команду
›

Это самый практичный модуль — делаем то, что вы уже делали вручную, но автоматически. У вас есть JSON-файлы с курсами из Telegram — теперь Claude Code читает их и генерирует полноценные HTML-страницы по шаблону. Меняется содержимое JSON → запускаешь скрипт → все страницы обновляются автоматически. Разбираем как описать шаблон страницы через промпт, как передавать данные и как проверять результат. После этого модуля добавление нового курса займёт у тебя 2 минуты.

Чтение JSON Генерация HTML Шаблонизация Массовое создание страниц
03
Локальный сервер — разворачиваем проект у себя
Тестируем сайт локально перед публикацией, работаем как разработчики
›

Локальный сервер — это когда сайт работает у тебя на компьютере по адресу localhost:3000, и ты видишь изменения мгновенно без деплоя. Claude Code поднимает его одной командой. Разбираем как это работает: Node.js запускает маленький веб-сервер, он отдаёт твои HTML-файлы в браузер. Учимся работать с hot reload — страница обновляется сама при изменении файла. Это профессиональный рабочий процесс: сначала тестируешь локально, потом деплоишь. Также разбираем как с помощью Claude Code дорабатывать файлы — просишь изменить цвет, добавить блок, исправить текст — и он сам редактирует нужные файлы.

localhost Node.js сервер Hot reload Редактирование файлов
04
Автодеплой — публикация одной командой
Git + GitHub + Netlify CLI = сайт обновляется без drag & drop
›

Пора забыть про перетаскивание папки в Netlify. Настраиваем полноценный pipeline деплоя: Git для версионирования (Claude Code объясняет и выполняет все команды), GitHub как удалённое хранилище, Netlify CLI для публикации прямо из терминала. После настройки процесс выглядит так: ты говоришь Claude Code «задеплой изменения» → он делает git add, git commit, git push и netlify deploy — сайт обновлён. Разбираем как откатить изменения если что-то пошло не так. Это профессиональный workflow которым пользуются все разработчики.

Git GitHub Netlify CLI Автодеплой Откат изменений
05
Автопилот — скрипты которые работают без тебя
Сайт обновляется сам, контент генерируется автоматически
›

Финальный и самый мощный модуль. Строим систему которая работает без твоего участия. Сценарий 1: добавляешь новый JSON-файл с курсом в папку → скрипт автоматически создаёт страницу и деплоит её на сайт. Сценарий 2: скрипт по расписанию проверяет Telegram-канал, берёт новые посты и обновляет раздел блога. Сценарий 3: один скрипт пересоздаёт весь сайт из исходников — полезно когда меняешь дизайн. Разбираем как использовать Claude Code для написания этих скриптов через диалог — описываешь что должен делать скрипт, он пишет код, ты запускаешь.
