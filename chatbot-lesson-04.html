Интеграции и промпт-инжиниринг

В этом уроке бот перестаёт быть простым автоответчиком. Подключаем Google Таблицы для сбора заявок и ChatGPT для умных ответов на любые вопросы.

Что такое интеграция и зачем она нужна

Интеграция — это когда бот обменивается данными с другим сервисом. Именно за интеграции бизнес платит больше всего.

Примеры интеграций

• Клиент оставил заявку → данные автоматически в Google Таблице
• Клиент оплатил → бот получил уведомление и открыл доступ
• Клиент написал вопрос → ChatGPT сгенерировал умный ответ
• Бот проверил наличие товара в базе → ответил есть ли нужная позиция

Интеграция с Google Таблицами

Самая популярная интеграция для малого бизнеса — сохранение заявок в Google Таблицу. Настраивается в Salebot через блок «Действие → HTTP-запрос».

Схема работы:

  1. Пользователь вводит имя → бот сохраняет в переменную name
  2. Пользователь вводит телефон → бот сохраняет в переменную phone
  3. Бот отправляет данные в Google Таблицу через Google Apps Script
  4. Строка с именем, телефоном и датой появляется в таблице автоматически
Инструмент

Используйте Make (integromat.com) для интеграций если не хотите разбираться с Google Apps Script. Make соединяет Salebot и Google Таблицы визуально без кода.

Подключаем ChatGPT к боту

Это самая мощная функция — бот с ChatGPT может ответить на любой вопрос клиента, даже тот который вы не предусмотрели в сценарии.

Для подключения нужен API-ключ OpenAI (openai.com → API). В Salebot добавьте блок «HTTP-запрос» и настройте обращение к API ChatGPT.

Промпт-инжиниринг для ботов

Чтобы ChatGPT отвечал правильно — нужен грамотный промпт. Промпт — это инструкция которую вы даёте нейросети.

Плохой промптХороший промпт
«Ответь на вопрос пользователя»«Ты консультант салона красоты Bella в Алматы. Отвечай только на вопросы об услугах и записи. Говори вежливо. Если вопрос не связан с салоном — вежливо откажи.»

Хороший промпт содержит:

  • Роль — кем является ИИ
  • Контекст — информация о бизнесе
  • Ограничения — о чём можно и нельзя говорить
  • Стиль — как именно отвечать
✏️ Задание к уроку
    1. Создайте Google Таблицу для сбора заявок — колонки: Имя, Телефон, Дата, Источник
    2. Подключите таблицу к боту из урока 3 через Make или Google Apps Script
    3. Напишите промпт для ChatGPT под вашу нишу — с ролью, контекстом и ограничениями
    4. Протестируйте — отправьте заявку через бота и убедитесь что данные появились в таблице
